# FCP AutoCut 호환성 체크리스트

테스터 로그·피드백으로 확인된 환경별·프로젝트 유형별 동작 상태를 기록한다.
로그를 받으면 **앱 버전 / macOS / FCP 버전 / 프로젝트 유형**을 확인해 아래 표를 갱신하고,
문제가 있으면 "문제점" 항목에 원인과 대응 버전을 기록한다.

범례: ✅ 정상 확인 · ⚠️ 조건부/임시방법 · ❌ 미지원(예정) · ❓ 미확인

## 로그 확인 절차 (테스터 문제 접수 시)

1. 로그 상단 헤더에서 **앱 버전 / macOS / FCP 버전 / 칩 / 모델 상태 / 미디어 실존 / XML 클립 구성** 확인
2. 최신 버전이 아니면 → 업데이트 안내 (github.com/groundroot/fcp-autocut/releases/latest)
3. 최신인데도 문제면 → 위 표에 유형·증상 기록하고 원인 분석 → 수정 후 해결 버전 기입
4. 미지원 유형(컴파운드/멀티캠)이면 → Break Apart 임시 방법 안내
