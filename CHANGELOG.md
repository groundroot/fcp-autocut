# FCP AutoCut 버전 업데이트 로그

## beta v0.6 — 2026-07-04

### 개선
- **모델 다운로드 진행 표시** — 다운로드 중인 모델 이름("Qwen3 음성 인식 모델" 등),
  진행률 바 + %, 다운로드 속도(MB/s), 몇 번째 모델인지(1/2) 표시.
  실제 받은 바이트 기준이라 네트워크가 느려도 정확하게 반영됨
- **멀티캠 프로젝트 안내** — 멀티캠(mc-clip) 프로젝트를 넣으면 명확한 안내와 임시 해결책
  (Break Apart Clip Items) 표시. 멀티캠 정식 지원은 v0.7 예정

## beta v0.5 — 2026-07-04

### 새 기능
- **자막 빈틈없이 채우기** 옵션 (기본 켬) — 끄면 실제 말하는 구간에 맞게
  자막 길이 조정 (짧은 끊김 메움 없이 순수 발화 타이밍, `--no-gap-fill`)
- **실패 시 "로그 다운로드" 버튼** — 앱 버전·macOS·입력 파일·선택 옵션이 포함된
  전체 로그를 다운로드 폴더에 저장 (피드백 첨부용)

### 베타 정책
- 베타 사용 횟수 **30회 고정** (소진 시 "정식 출시를 기다려주세요" 안내, 구매 버튼 없음)
- 개발 빌드는 무제한 유지, 정식 빌드는 무료 데모 10회 + PRO 구매

## beta v0.4 — 2026-07-04

### 새 기능 (자동 커팅 — Auphonic Automatic Cutting 참고, 한국어 지원)
- **군소리 컷**: 홀로 발화된 추임새("어/음/아/흠" 등)와 기침·헛기침(음성 감지됐지만
  단어가 없는 짧은 구간)을 무음제거본 타임라인에서 잘라냄. 단어 정렬 타임스탬프 기반이라
  문장 속 진짜 단어("아 그렇군요"의 "아")는 건드리지 않음 — 앞뒤로 띄어진 단독 발화만 컷
- **다시 말하기 컷**: (1) 즉시 반복("우리가, 우리가") (2) 문장을 끝맺지 못하고 비슷한
  문장으로 다시 시작한 경우 미완성 쪽을 잘라내고 완성된 문장만 남김.
  잘린 단어는 자막에서도 함께 제외됨
- 두 기능 모두 무음컷 아래 체크박스 (기본 켬), CLI `--cut-fillers` / `--cut-retakes`

### 호환성
- **FCPXML 버전 호환 (FCP 10.x ~ 12.3)**: 구버전 추출본(DTD 1.8 미만)은 자동으로 1.8로
  승격해 caption 요소가 DTD 검증을 통과하도록 함. 아주 오래된 `<clip>` 구조(2018 이전)는
  친절한 안내 메시지와 함께 재내보내기 유도

### 용량 (베타 DMG 43% 절감)
- VAD의 torch 의존 제거 — silero ONNX 모델(2.2MB)을 내장하고 onnxruntime으로 실행.
  torch·torchaudio·sympy 등 약 400MB 제거, 번들 런타임 950MB → 543MB.
  개발 venv는 기존 torch 경로 유지 (결과 차이 ≤ 0.02초 검증 완료)

### UI/UX
- 진행 문구를 **순차 표시 큐**로 변경 — 진행률(%)은 실시간으로 움직이되,
  문구는 각각 1.8초씩 읽을 수 있게 하나씩 표시 (밀리면 오래된 문구부터 생략)

## beta v0.3 — 2026-07-04

### 변경
- **AI 문맥 교정 제거** — 앱을 단순하게 유지. 자막 품질의 핵심은 의미 단위 분할로 집중
  (silence-cutter의 `--polish` CLI 플래그는 남아 있으나 앱에서는 사용 안 함)
- **피드백 버튼** — 클릭 시 Fillout 피드백 폼(forms.fillout.com/t/1Hnsen7DLZus)을 브라우저에서 열도록 연결
  (기존 메일 초안 방식 및 FeedbackView 제거)

### 개선 (의미 단위 분할)
- 존칭 관형형(하신/되신/주신/가신/오신/계신)을 분할 경계로 인식 —
  "…걷기 하신 | 하나님께 영광 돌립니다"처럼 길이 초과 시 자연스러운 지점에서 나뉨
- "짧게" 프리셋에서 4~5자 단편 자막이 생기던 문제 수정 —
  남은 꼬리가 너무 짧으면 자르지 않고 이어붙임 (약간의 길이 초과 허용)
- 세 가지 길이 프리셋(18/27/38자) 전부에서 의미 단위 보존 검증 완료

## beta v0.2 — 2026-07-04

### 새 기능
- **글자 크기** 옵션: 작게(32pt) / 보통(42pt) / 크게(54pt) / 아주 크게(68pt) — 타이틀 자막 폰트
- **자막 길이** 옵션: 짧게(한 줄 18자) / 보통(27자) / 길게(38자) — VREW의 글자 수 지정 방식 참고,
  길이와 무관하게 분할은 항상 문맥(의미 단위) 우선
- **AI 문맥 교정** 토글: 로컬 LLM(Qwen2.5-3B)이 오인식·구어체를 문맥에 맞게 다듬음.
  타이밍·의미 불변, 처리 시간 증가. (silence-cutter에 `--polish` 신설, 교정 3순위로 통합)

### UI/UX
- 실행 중 화면을 로그 대신 **프로그레스 바 + doodling 스타일 한글 문구**로 교체
  ("소리 안 들리는 구간을 사시미로 써는 중… 🔪", "자막을 노릇노릇 굽는 중… 🔥" 등)
- 음성 인식 구간은 실제 진행률(i/total)로 바가 움직임
- 원본 로그는 "자세한 로그" 접이식으로 이동, 실패 시에만 자동 표시

### 수정
- 개발 빌드에서 리빌드할 때마다 키체인 암호를 묻던 문제 수정 — 사용 카운터의 Keychain 저장은
  서명이 고정된 앱스토어 정식 빌드에서만 사용 (개발/베타는 UserDefaults)
- 데모 10회 제한을 앱스토어 정식 빌드 전용으로 변경 — 개발·베타 빌드는 카운터 없이 무제한

## beta v0.1 — 2026-07-04

첫 베타 버전.

### 기능
- 파이널컷 프로젝트 드래그앤드롭 입력 (FCP에서 직접 드래그 또는 .fcpxmld/.fcpxml 파일)
- 무음컷: Silero VAD 기반 무음 구간 제거, 컷 강도 4단계 (약하게/보통/강하게/매우 강하게)
- 자막 자동화: Qwen3-ASR(1.7B) + ForcedAligner 음성 인식, 의미 단위 우선 자막 분리
- 자막 형식 선택: 타이틀(기본) / 캡션 / 둘 다
- 결과물을 파이널컷으로 바로 드래그해서 import (FCPXML 페이스트보드 지원)
- AI 모델 자동 감지 + 미설치 시 앱 내 다운로드 (약 3.5GB)

### 베타
- 베타 빌드는 사용 제한 없음 (데모 10회 제한은 앱스토어 정식 빌드 전용)
- 앱 내 "피드백" 버튼 — 앱 버전/macOS 정보 자동 첨부, 메일로 전송 (webhook 연동 가능)
- 베타 배포 DMG 패키징: 번들 Python 런타임 + 정적 ffmpeg 포함 (다른 Mac에서 자기완결 실행)

### 유료화 (정식 빌드)
- 무료 데모 10회 (성공한 프로젝트 추출 기준, 재설치해도 유지)
- 소진 후 PRO 영구버전 구매 (StoreKit 2, App Store Connect 등록 예정)

### 빌드
- `./build_app.sh` 개발용 / `appstore` MAS 구조 / `dmg` 베타 배포 DMG
- `./build_runtime.sh` 자기완결 런타임 생성 (~865MB)

### 알려진 제한 (앱스토어 출시 전 해결 예정)
- 앱스토어 빌드에 ffmpeg 미포함 — AVFoundation 오디오 추출로 대체 예정
- 샌드박스 미디어 폴더 접근 권한 UX 미구현
- 앱 아이콘 없음
