# FCP AutoCut 버전 업데이트 로그

## beta v0.12.82 — 2026-07-18 (자막 카드 편집 · 파이널컷 타이틀 · 스토리라인 · 이중언어)

### 새 기능
- **자막 카드 편집(CapCut식)** — 대본 편집기 「자막 보기」: 실제 출력과 동일한 자막 카드를
  번호순으로 보고 클릭 편집·**병합(⌫)·분리(⌘Enter)**·방향키 카드 이동. 길이·2줄을 툴바에서
  즉시 조절(카드 수 실시간 반영), 결과는 내보내기까지 반영
- **파이널컷 타이틀 지정** — 타임라인의 타이틀 클립을 드래그하거나 「고르기」 목록(FCP Titles
  브라우저와 동일 소스, 검색·카테고리)에서 선택하면 **그 타이틀 룩으로 자막 생성**
- **스토리라인으로 묶기** — 자막을 Create Storyline(보조 스토리라인)으로 그룹화. 화자 분리
  시 화자별 분리. 자막본·무음제거본 모두 지원
- **고정밀 화자분리(pyannote)** — 겹치는 발화 감지. 설정에 HuggingFace 토큰 등록 시 활성,
  기본 방식도 창 세밀화·라벨 스무딩으로 개선
- **구두점 바이블** — LLM 임의 삽입 폐지: `?`는 질문에만(문법 판정, ‘-세요’ 의문 지원),
  쉼표는 **실제 발화 쉼** 위치만. 음악/비발화 구간 **환청 자막 차단**(발화 게이트)
- **영어 UI + 한국어 완전 현지화** — 설정 › 일반에서 언어 전환(시스템/한국어/English)
- **MAS(앱스토어) 출시 준비** — CutPilot 빌드 파이프라인(샌드박스 서명·pkg), 자동(다국어)
  모드 지배 언어 고정(영어 영상 부분 번역 방지)

### 수정
- FCP 임포트 "not on an edit frame boundary" 근본 해결(타이틀 델타 격자)
- 화자 2명인데 타이틀 색 3개로 나오던 문제(미배정 자막 → 최근접 화자)
- UI/UX 정리 — 사이드바 계층화(고급 옵션 접힘)·색상 체계 3군 정리
- 앱 용량 축소 — 미사용 번들 라이브러리 정리로 DMG 약 63MB 감소
