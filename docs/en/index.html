Features

What it can do

Built for talk-heavy long-form — interviews, lectures, sermons, podcasts, YouTube. It takes over the hours you spent cutting silence and typing captions by hand.

✂️Silence Cut — trim only the dead air

Automatically finds and removes the gaps between phrases. Your original timeline stays untouched — you get a separate compressed version with the silence gone.

  • Cut strength (gentle to aggressive) controls how tightly it trims
  • The rest of your edit structure is left alone
  • Filler-word cut — removes "uh / um" and coughs (mild / medium / strong)

💬Auto Captions — exactly what was said

Local AI speech recognition (Qwen3-ASR) plus word-level forced alignment produces captions that stick to the speech.

  • Hears every audio item in the timeline — connected clips and connected storylines included
  • Splits on meaning — reads the whole sentence and breaks where it reads best
  • Punctuation with rules — question marks only on real questions, commas only on real pauses
  • No language drift — an English video gets English captions from start to finish
  • No captions over music — hallucinated captions are blocked by audio analysis

Transcript Editor

📝 Transcript Editor — edit video like a document

When analysis finishes, the [Edit Transcript] button opens a window that feels like a word processor. Read the transcript, delete what you don't need, and that span is cut from the timeline. Every edit is undoable with ⌘Z.

Like a document, keyboard-first

  • Drag to select · click to play from there · double-click to fix a word in place
  • Delete = cut — playback automatically skips deleted spans
  • Free cursor with arrow keys · repeated Backspace deletes · ESC leaves editing
  • Space plays from the cursor; clicking the waveform scrolls the text along
  • One-click Silence Cut and Filler Cut for the whole transcript

💬 Caption view — polish caption by caption (CapCut style)

  • One card = one caption exactly as it will export, numbered with timecodes
  • ⌫ at the start of a caption merges with the previous one · ⌘Enter splits at the cursor
  • Toolbar length slider + 2-line toggle reflows the whole set instantly
  • Your merges and splits carry through to the exported result

🤖 AI Agent — tell it, and it edits

It never rewrites your words — it only cuts what you ask, and every result can be reviewed and restored.

  • Summarize & extract — "keep the essentials" · "down to 5 minutes" · "keep only the sales talk"
  • Google SEO outline — builds an H2/H3 table of contents; delete whole sections from it
  • Punctuate — periods, question marks, and commas by codified rules
  • Offline by default — optionally connect OpenAI · Gemini · Claude API keys

Subtitle Style & Titles

🎨 Subtitle style & titles — finished from the start

Live preview

  • Adjust font size, caption length, 2-line captions, and position with instant feedback
  • Auto-detects aspect ratio (16:9 / 9:16); 9:16 shows YouTube · Instagram · TikTok safe-zone guides
  • What you see in the preview matches Final Cut's actual render 1:1
  • Mixed frame rates are safe — a 30p project with 29.97fps sources still aligns exactly

Your own titles, as-is

  • Drag in a Final Cut title, or pick one from the built-in browser → captions are generated in that title
  • Auto-scans installed titles (built-in + third-party), with a cache so the list opens instantly
  • No title swapping ever — captions are created with native values from the start, so the timeline never drifts
  • Supports FCP 12.3 Subtitle titles + the Subtitles role (select all, batch edit)

How to use

🔓 Seeing an "unidentified developer" warning? Do this once

The beta isn't Apple-notarized yet, so macOS blocks the very first launch. Allow it with the steps below — needed only once; after that it opens with a double-click.

1

Try opening it once

Double-click FCP AutoCut in Applications → when the "can't be opened" alert appears, click Done (OK). This attempt is what makes the next step's button appear.

2

Allow it in Settings

Go to System Settings → Privacy & Security, scroll down, and click Open Anyway next to "'FCP AutoCut' was blocked."

3

Enter your password and open

Enter your admin password (or Touch ID) and click Open. From then on it launches without any warning.

💡 This warning doesn't mean anything is wrong with the app — it's macOS's standard notice for betas that haven't gone through Apple's notarization yet. If you get stuck, open an issue and we'll help right away.
