Why FCP AutoCut

믿고 맡길 수 있는 이유

⚡

내보내기 과정이 없다 — 프로젝트를 그대로 드래그

다른 도구는 음성 인식을 위해 영상을 렌더링하거나 오디오를 따로 추출해서 넣어야 하죠. FCP AutoCut은 그 과정이 통째로 사라집니다. 편집하던 프로젝트를 드래그앤드롭하면 컷편집된 타임라인의 오디오를 그대로 분석해 — 자막 생성부터 Text-Based Editing(대본 기반 편집)까지 바로 이어집니다.

🔒

100% 내 Mac에서

음성 인식·자막·편집·AI 정리까지 전부 온디바이스. 영상·음성이 밖으로 나가지 않습니다.

♻️

완전 비파괴

결과는 항상 새 프로젝트로. 어저스트먼트 레이어·B롤·보조 스토리라인·컷 구조를 그대로 보존합니다.

🎬

파이널컷에 딱 맞게

자막이 처음부터 FCP 네이티브 타이틀로 생성돼 교체가 필요 없고, 혼합 프레임레이트에서도 어긋나지 않습니다.

👀

보이는 그대로

앱에서 본 자막 크기·위치가 파이널컷 실제 렌더와 1:1로 일치합니다.

Features

무엇을 할 수 있나요

인터뷰 · 강의 · 설교 · 팟캐스트 · 유튜브 같은 말이 많은 롱폼을 위한 도구입니다. 침묵을 일일이 자르고, 자막을 손으로 치던 그 시간을 대신합니다.

✂️무음컷 — 침묵만 걷어내기

말과 말 사이의 빈 구간을 자동으로 찾아 잘라냅니다. 원본 타임라인은 그대로 두고, 무음을 제거한 압축된 버전을 따로 만들어 줍니다.

  • 컷 강도(약~매우 강)로 얼마나 촘촘히 자를지 선택
  • 나머지 편집 구조는 손대지 않음
  • 군소리 컷 — "어/음/아" 추임새·기침 제거(약/중/강)

💬자막 자동화 — 말하는 대로, 정확하게

로컬 AI 음성 인식(Qwen3-ASR) + 단어 단위 정밀 타이밍으로 말에 딱 붙는 자막을 만듭니다.

  • 타임라인의 모든 오디오 인식 — 커넥트 클립·연결 스토리라인까지
  • 의미 단위 분리 — 문장 전체를 보고 가장 읽기 좋은 곳에서 끊음
  • 구두점도 기준 있게 — 물음표는 진짜 질문에만, 쉼표는 실제 쉼에만
  • 언어가 흔들리지 않게 — 영어 영상은 처음부터 끝까지 영어 자막
  • 음악 구간엔 자막 없음 — 환청 자막을 오디오 분석으로 차단

Transcript Editor

📝 대본 편집기 — 영상을 '글'처럼 편집

분석이 끝나면 [대본 편집] 버튼으로 문서 편집기 같은 창이 열립니다. 대본을 읽으며 필요 없는 부분을 지우면, 그 구간이 타임라인에서 잘려 나옵니다. 모든 편집은 ⌘Z로 복원.

문서처럼, 키보드만으로

  • 드래그 선택 · 클릭 재생 · 더블클릭으로 단어 즉시 수정
  • 지우면 잘린다 — 재생 시 삭제 구간 자동 스킵
  • 방향키 자유 커서 · 백스페이스 연속 삭제 · ESC 편집 탈출
  • Space = 커서 위치부터 재생, 파형 클릭 시 글도 따라 스크롤
  • 무음컷·군소리 삭제 일괄 버튼

💬 자막 뷰 — 자막 단위로 다듬기 (CapCut 스타일)

  • 실제 출력될 자막 한 장 = 카드 한 장, 번호·타임코드 표시
  • 자막 맨 앞에서 ⌫ = 앞 자막과 병합 · ⌘Enter = 커서에서 분리
  • 툴바 길이 슬라이더 + 2줄 토글로 자막 수 즉시 조절
  • 병합·분리한 경계는 출력 결과에 그대로 반영

🤖 AI 에이전트 — 말하면 대신 편집

텍스트는 절대 바꾸지 않고 필요한 부분만 잘라내며, 결과는 언제든 검토·복원할 수 있습니다.

  • 요약 & 발췌 — "핵심만" · "5분으로" · "매출 얘기만 남겨줘"
  • Google SEO 구조 정리 — H2·H3 목차 생성, 섹션 통째 삭제
  • 구두점 넣기 — 마침표·물음표·쉼표를 명문화된 규칙대로
  • 기본은 오프라인 — 원하면 OpenAI·Gemini·Claude API 키 연결

Subtitle Style & Titles

🎨 자막 스타일 & 타이틀 — 처음부터 완성형

실시간 프리뷰

  • 글자 크기·자막 길이·2줄 자막·위치를 즉시 확인하며 조절
  • 화면비(16:9/9:16) 자동 감지, 9:16엔 유튜브·인스타·틱톡 세이프존 가이드
  • 프리뷰에서 본 크기·위치가 파이널컷 실제 렌더와 1:1
  • 혼합 프레임레이트 안전 — 30p 프로젝트 + 29.97fps 소스도 정확히 정렬

내가 쓰는 타이틀 그대로

  • 파이널컷 타이틀을 드래그하거나 "고르기" 목록에서 선택 → 자막이 그 타이틀로 생성
  • 내장 + 사용자 설치 타이틀 자동 스캔, 캐시로 목록 즉시 오픈
  • 교체가 필요 없는 타이틀 — 처음부터 네이티브 값으로 생성돼 타임라인이 어긋날 일 없음
  • FCP 12.3 Subtitle 타이틀 + Subtitles 롤 지원(전체 선택·일괄 편집)

How to use

사용법 — 3단계

1

넣기

파이널컷에서 프로젝트를 잡아 앱의 점선 박스로 드래그

2

실행

원하는 옵션을 켠 뒤 실행. 진행 상황과 남은 시간이 표시됩니다

3

돌려받기

결과 파일을 잡아 파이널컷으로 드래그 → 새 프로젝트로 import (원본 비파괴)

처리한 프로젝트는 작업 히스토리(최대 50건)에서 결과 폴더를 바로 열 수 있어요.

Requirements

설치 & 요구 사항

MacApple Silicon (M1 이상) — Intel 미지원
macOS14 (Sonoma) 이상
Final Cut Pro10.x ~ 12.3
디스크최초 1회 AI 음성 모델 약 3.5GB

DMG를 열고 앱을 Applications로 드래그하면 설치 끝. 설치 후에는 앱 안의 "지금 업데이트" 버튼으로 이후 버전을 바로 받을 수 있어요. 성능·개인정보: 모든 AI 처리는 온디바이스로, 네트워크 연결 없이 동작합니다.

🔓 "확인되지 않은 개발자" 경고가 뜬다면 — 한 번만 이렇게

베타 버전은 아직 Apple 공증(notarization) 전이라, 처음 열 때 macOS가 실행을 한 번 차단합니다. 아래 순서대로 허용해 주세요. 최초 1회만 필요하고, 이후에는 더블클릭으로 바로 열립니다.

1

일단 한 번 실행

Applications에서 FCP AutoCut을 더블클릭 → "열 수 없습니다" 경고가 뜨면 완료(확인)를 누릅니다. 이 시도가 있어야 다음 단계 버튼이 나타나요.

2

설정에서 허용

시스템 설정 → 개인정보 보호 및 보안으로 가서 아래로 스크롤 → "'FCP AutoCut'이(가) 차단되었습니다" 옆의 그래도 열기를 클릭합니다.

3

암호 입력 후 열기

관리자 암호(또는 Touch ID)를 입력하고 열기를 누르면 실행됩니다. 다음부터는 경고 없이 바로 열려요.

💡 이 경고는 앱에 문제가 있어서가 아니라, 베타가 아직 Apple의 공증 절차를 거치지 않았기 때문에 나오는 표준 안내입니다. 막히는 부분이 있으면 베타 오픈채팅에서 바로 도와드려요.
